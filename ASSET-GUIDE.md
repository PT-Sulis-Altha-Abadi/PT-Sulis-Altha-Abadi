## Aturan Penamaan

- Gunakan huruf kecil semua.
- Gunakan tanda minus, bukan spasi.
- Contoh benar: `telecommunication-hero.jpg`
- Hindari nama seperti `foto baru final FIX.png`
- Untuk logo, pilih `svg` jika tersedia agar tetap tajam.
- Untuk foto, gunakan `jpg`, `jpeg`, `png`, atau `webp`.
- Jika mengganti aset yang sudah dipakai, paling mudah gunakan nama file yang sama agar tidak perlu mengubah kode.

## Ukuran Yang Disarankan

| Jenis aset | Ukuran yang disarankan |
| --- | --- |
| Hero banner | minimal `1600 x 700 px` |
| Foto pimpinan | minimal `800 x 1100 px` |
| Foto produk | minimal `700 x 500 px` |
| Foto kartu layanan | minimal `900 x 700 px` |
| Logo | latar transparan, lebih baik format `svg` |
| Bendera | rasio sekitar `3:2` atau `4:3` |
